# Docs

هذا المجلد يحتوي على التوثيق الداخلي للمشروع والمراجعات المرحلية.

## المستندات المتاحة

- `phase-1-review.md`: مراجعة تنفيذ المرحلة الأولى مقارنة بالخطة الموجودة في `PROJECT_PLAN_LARAVEL.md`.
- `phase-2-implementation.md`: توثيق تفصيلي لكل ما تم تنفيذه في المرحلة الثانية، مع شرح القرارات والملفات والعلاقات بين الأجزاء.
- `phase-4-design-tool.md`: توثيق أداة التصميم — Fabric.js، الـ Canvas Engine، وبنية الـ UI.
- `phase-4-responsive-layout.md`: التوثيق الأولي للـ responsive layout (قبل التحديث الكامل).
- `design-tool-ui-polish.md`: **(الأحدث)** توثيق شامل لكل إصلاحات الـ UI والـ Responsive — إصلاح الفجوة بين الهيدر والمحتوى، الـ scroll في Sidebar، خط Tajawal، تمركز العنوان، وجميع الـ breakpoints للتابلت والموبايل بكل أحجامه.

## ملاحظات

- هذا التوثيق يعكس حالة المشروع الحالية وقت المراجعة.
- تم تحديث مراجعة المرحلة الأولى بعد إغلاق فجوات الاعتماديات و`Seeder` وأيقونات الـ PWA المؤقتة، واعتماد المرحلة الأولى كمكتملة.
- عند تنفيذ أي جزء جديد من المرحلة الأولى أو المراحل اللاحقة، يُفضل تحديث المستندات هنا حتى تبقى مرجعًا دقيقًا.
